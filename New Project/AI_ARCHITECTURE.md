## Message Types

```
Ask Mode Message:
{
  id: number,
  role: 'user' | 'assistant' | 'error',
  content: string (markdown),
  timestamp: ISO string,
  metadata: {
    provider: string,
    model: string,
    tokens: number
  }
}

Agent Mode Message:
{
  id: number,
  role: 'user' | 'agent' | 'error',
  content: string,
  timestamp: ISO string,
  metadata: {
    filesCreated: string[],
    filesModified: string[],
    details: object
  }
}

Agent Log:
{
  type: 'info' | 'success' | 'error' | 'warning' | 'terminal',
  message: string,
  details?: string,
  timestamp: ISO string
}
```
